## Como executar

Clone o projeto e acesse a pasta.

```bash

$ git clone https://github.com/MarcioJnr/Perc-App-PWA
$ cd Perc-App-PWA

```

Para iniciá-lo, siga os passos abaixo:

```bash

# Instalar as dependências com npm
$ npm install

# Iniciar o projeto com npm
$ npm start

```
